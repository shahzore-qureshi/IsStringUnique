# Is String Unique?
Simple implementation of a String uniqueness evaluator in Java.

## Usage
Type in the following commands in a terminal:
(note: make sure that asterisk * is escaped via quotes)

```java
javac ICM.java
java ICM test 
>>> String is not unique.

javac ICM.java
java ICM tes 
>>> String is unique.

javac ICM.java
java ICM tesla 
>>> String is unique.

javac ICM.java
java ICM tessla 
>>> String is not unique.

```
